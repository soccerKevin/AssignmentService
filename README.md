# Student Assignments
A simple app to manage students and courses

## Setup
### dev
- yarn install
- docker compose up
- yarn run dev

### prod
- yarn run all

## API
- student/:id (Get)
- student     (Post)
- student/:id (Put)
- student/:id/courses (Get)
- student/:id/grade/average (Get)
- student/:studentId/course/:courseId/grade (Get)

- registration (Post)
- registration/update (Put)
- registration (Delete)

- course/:id/students (Get)
- course (Post)
- course/:id (Put)
